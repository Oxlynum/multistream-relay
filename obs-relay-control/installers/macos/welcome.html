Relay Control for OBS

This plugin adds a Relay Control dock to OBS Studio that lets you manage your HEVC cloud transcoding relay server without leaving OBS.

  • Pick your GPU cloud provider (Vultr, RunPod, DigitalOcean, AWS, and more)
  • Enter your server IP — OBS's stream URL is set automatically
  • Configure Twitch, Kick, and YouTube stream keys and bitrates
  • The relay starts and stops automatically with OBS's stream button

OBS Studio must already be installed. After installation, restart OBS and open Docks → Relay Control.
