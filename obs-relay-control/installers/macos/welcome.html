SlimCast for OBS

This plugin adds a SlimCast dock to OBS Studio so you can stream to multiple platforms simultaneously — no setup, no config files, no terminal.

  • Enter your SlimCast API key once — OBS is configured automatically
  • Click Start Streaming in OBS — your streaming server starts and all platforms go live
  • Credits balance and warnings shown directly in OBS

OBS Studio must already be installed. After installation, restart OBS and open Docks → SlimCast.

Sign up at slimcast.com to get your API key.
